# Helper Utilities

A collection of helpful developer utility functions.

## Features

- Format JSON output
- Generate UUIDs
- Convert timestamps between formats
- Base64 encode/decode strings

## Usage

Ask your AI agent to use these utilities for common development tasks like formatting data, generating identifiers, or converting between formats.
